Create a program that generates all cards of a card playing deck. First print the clubs, starting from the Ace, ending 
with a King. Continue with the same cards from Hearts, Diamonds and Spades. Print them in the format given below.
Note

Try using the enumeration types you have created in the previous problems.

Example input:
Card Deck

Output:
Ace of Clubs
Two of Clubs
Three of Clubs
Four of Clubs
Five of Clubs
...
...
King of Spades
